# 👥 Users App

Aplicación web que consume una API para mostrar usuarios.

## 🚀 Demo
https://jbfrankmorrison.github.io/users-app/

## ✨ Funcionalidades
- Consumo de API (fetch)
- Búsqueda en tiempo real
- Loader (estado de carga)
- Manejo de errores
- Visualización de usuarios

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
